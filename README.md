# consize

Change buffer size of Command Prompt

## Usage

### reset cols/rows in default you set.
```
consize
```

### reset cols and set rows.

```
consize 25
```

### eset cols and rows.

```
consize 25 80
```

## Installation

```
go get github.com/mattn/consize
```
